Подсистема УВО

Предназначены для решения следующих задач:

  • управление, контроль и диагностика дренажной системы (в том числе насосы осушения проточной части);
  • управление лекажным агрегатом;
  • управление системой отжатия воды для режима СК;
  • управление двигателями и перепускными клапанами насосов МНУ;
  • управление и контроль системы охлаждения (ТВС);
  • управление и контроль системы торможения и подъема гидроагрегата;
  • управление и контроль системы пожаротушения.

Общее техническое решение

Местные шкафы управления предоставляют возможность как местного ручного оперирования двигателями и приводами, так и автономную работу при неисправности связи с АСУ ТП или выводе связанных элементов АСУ ТП в ремонт (кроме насосов маслонапорной установки).

Для каждого электродвигателя и привода предусмотрены устройства плавного пуска (УПП) и рубильники-разъединители (кроме системы МНУ, где применяются разъединители), обеспечивающий видимый разрыв и возможность фиксации навесным замком для соблюдения регламента и обеспечения безопасности работ.

Для ручного управления, на дверях шкафов предусмотрена световая сигнализация о состоянии двигателей и приводов, показания с соответствующих датчиков. Показания аналоговых датчиков выведены через цифровые индикаторы. Ключи выбора режима работы и приоритета выполнены на кулачковых переключателях с фиксацией.

Сигналы с соответствующих датчиков поступают как в местных шкаф управления для обработки, так и в АСУ ТП. Подключение может производиться параллельно (последовательная «токовая петля»), либо местный шкаф управления выступает в качества шлюза.

По желанию заказчика выбирается один из двух способов автоматизации:

  • программируемое промышленное реле с цифровым интерфейсом для связи с АСУ ТП (Modbus RTU через RS485) и предоставлением средства программирования для инженерного персонала;
  • релейная схема с использованием промышленных реле и взаимодействием с АСУ ТП через физические сигналы.

Внутри шкафа предусмотрено внутреннее освещение и розетка 220VAC.

Силовые шкафы МНУ

Система представлена одним шкафов для двух насосов. Исполнение шкафов – напольное с прокладкой кабелей снизу. Для защиты электродвигателей применяются откидывающиеся плавкие вставки (плавкие вставки входят в комплект ЗИП). УПП отдельно подключаются к АСУ ТП по полевому интерфейсу (протокол Modbus RTU через RS485) для получения более полной информации о состоянии электродвигателя.

Пример силового шкафа МНУ

При ручном управлении с лицевой панели, существует возможность оперирования перепускным клапаном насоса при их соответствующем включении и отключении, что позволяет осуществлять безопасную автономную ручную работу.

На двери каждого шкафа находятся следующие элементы:

  • ключ «Автоматический – Отключен – Включен);
  • сигнальные лампы о наличии неисправности УПП (электродвигателя), наличия ввода, наличия внутреннего питания 24VDC;
  • сигнальные лампы «Насос включен», «Насос отключен»;
  • амперметр тока нагрузки;

Лекажный агрегат

Система представлена одним навесным шкафом для управления двумя насосами. Ввод кабелей возможен сверху и снизу.

В шкаф заводятся сигналы как с аналогового датчика, так и конечных выключателей уровня. Сигналы с датчиков резервируют друг друга. Это повышает надежность системы в случае отказа связи или блокировки поплавка. В автоматическом режиме работы насосы запускаются поочередно.

Пример монтажа шкафа лекажного агрегата

На двери шкафа находятся следующие элементы:

  • Совмещенные ключи выбора режима работы и приоритета (Автоматический– Отключен – Включен) для каждого из насоса.
  • сигнальные лампы о наличии неисправности УПП (электродвигателя) для каждого из насоса, наличия ввода, наличия внутреннего питания 24VDC;
  • сигнальные лампы «Насос включен», «Насос отключен для каждого из насоса;
  • цифровой индикатор уровня от аналогового датчика.

Дренажный агрегат (откачка воды с крышки турбины)

Система представлена одним навесным шкафом для управления двумя насосами. Ввод кабелей возможен сверху и снизу.

В шкаф заводятся сигналы как с аналогового датчика, так и конечных выключателей уровня. Сигналы с датчиков резервируют друг друга. Это повышает надежность системы в случае отказа связи или блокировки поплавка. Возле дренажных насосов устанавливаются компактные пульты управления высокой защиты для дополнительных возможностей ручного управления. В автоматическом режиме работы насосы запускаются поочередно.

Пример монтажа шкафа дренажного агрегата

На двери шкафа находятся следующие элементы:

  • Совмещенные ключи выбора режима работы и приоритета (Автоматический – Отключен – Включен) для каждого из насоса.
  • сигнальные лампы о наличии неисправности УПП (электродвигателя) для каждого из насоса, наличия ввода, наличия внутреннего питания 24VDC;
  • сигнальные лампы «Насос №1 включен», «Насос №1 отключен», «Насос №2 включен», «Насос №2 отключен»;
  • цифровой индикатор уровня от аналогового датчика.

Автоматика ТВС гидроагрегата

Система представлена одним навесным шкафом для управления тремя задвижками. Ввод кабелей возможен сверху и снизу.

Управление задвижками производится вручную в местном режиме, либо по сигналу от АСУ ТП. Отключение привода по достижения концевым или моментным выключателем упорного положения производится автоматически. В ручном режиме управление может быть импульсным, либо требующим постоянного нажатия на кнопку управления (выбор осуществляется ключом внутри шкафа).

Пример лицевой панели шкафа автоматики ТВС гидроагрегата

На двери шкафа находятся следующие элементы:

  • совмещенные ключи выбора режима работы (Автоматический – Ручной) для каждого из привода;
  • кнопки открытия и закрытия для каждой из задвижки;
  • сигнальные лампы о наличии неисправности УПП (электродвигателя) для каждого из привода, наличия ввода, наличия внутреннего питания 24VDC.
